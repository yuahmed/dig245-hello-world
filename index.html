  5. Chosen learn-computing activity: NETWORKS.

    This activity provides information about networks and explains how users are connected to the internet, and how the internet is connected. An IP address is used to identify devices connected to the internet and is unique. Currently, IPv6 is in use. A domain name is the human-readable equivalent of an IP address and browsers have the functionality to perform a DNS lookup to find the corresponding IP address and fetch data accordingly. The activity then explains some useful CLI terminology - for example, upon using the ping command I saw that my laptop connected to google.com in ~11ms, to davidson.edu in ~4ms and to amazon.com in ~24ms. I think this is because of my distance to the servers of each domain physically - Davidson is fastest since I am on campus. Meanwhile, the netstat -at command showed that I had more than 20+ connections to/from my laptop.
